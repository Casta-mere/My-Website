# Link

React 超链接小组件

[这里](https://castamerego.com/docs/Snippets/Components/Link) 介绍了这个小组件

## Docusaurus 超链接

可直接集成到 Docusaurus MDX 中

## 功能对比

| 功能 \\ 组件         | Link          | DocusaurusMDXLinkEnhance |
| -------------------- | ------------- | ------------------------ |
| 各种动画             | ✔️             | ✔️                        |
| 设置是否显示悬浮框   | ✔️(默认不显示) | ❌(默认显示)              |
| Docusaurus 优化      | ❌             | ✔️                        |
| 跳转逻辑自定义       | ✔️             | ❌                        |
| 字体字号下划线自定义 | ✔️             | ❌                        |
| 添加自定义颜色       | ✔️             | ❌                        |
